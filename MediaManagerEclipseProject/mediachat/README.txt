
Projeto Eclipse: MediaManagerEclipseProject

Estrutura:
- src/ (código fonte Java)
- test/ (testes JUnit)
- data/ (arquivos .tpoo gerados em tempo de execução)

Para importar no Eclipse:
1. File -> Import -> General -> Existing Projects into Workspace OR
   criar um novo Java Project e apontar as pastas src e test como source folders.
2. Certifique-se de adicionar JUnit 4 ao classpath dos testes.

OBS: Os arquivos .tpoo são objetos serializados em data/.
